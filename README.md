# Gen Z Career Aspirations — KultureHire Milestone 3

A structured survey form built using the **5W1H problem statement framework** 
to explore why Gen Z feels unhappy, confused, or dissatisfied with jobs.

## 📋 About This Project

This form was created as part of **KultureHire's Milestone 3** task focused on 
documenting the career aspirations gap between Generation Z employees and employers.

## 🧩 Framework Used

The form is structured around the **5W1H model**:

| Section | Focus |
|--------|-------|
| **WHAT** | What Gen Z expects vs. what employers offer |
| **WHY** | Root causes of career dissatisfaction |
| **WHO** | Who is most affected |
| **WHEN** | At what career stage does dissatisfaction peak |
| **WHERE** | Work environment preferences |
| **HOW** | Motivations and solutions |

## 👥 Dual Perspective Design

Every major section captures **both sides**:
- 🟢 Gen Z's perspective — what they feel and want
- 🟠 Employer's perspective — what they observe and struggle with

## 🛠️ Built With

- Pure HTML, CSS, JavaScript
- No frameworks or external dependencies
- Google Fonts (Syne + DM Sans)
- Fully responsive — works on mobile and desktop

## 🚀 How to Use

1. Clone or download this repository
2. Open `gen_z_career_form.html` in any browser
